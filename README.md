
# Map Maker
This is mostly an excuse to play with Vercel and NextJS.  We'll see where it goes.

# Technologies
- NextJS
- Vercel
- Auth0

# How To's

## Run the client
```
npm run dev
```

Go to http://localhost:3000

## Run migrations

`npm run migrate-all`
